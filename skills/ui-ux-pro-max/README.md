# UI/UX Pro Max

A Claude skill that provides AI-powered design intelligence with 50+ UI styles, 161 color palettes, 57 font pairings, and 99 UX guidelines across 10 technology stacks.

> Original skill by [NextLevelBuilder](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill) — included in this collection under MIT license.

## What the skill does

Provides a comprehensive, priority-ranked design reference covering:

- **Accessibility** (CRITICAL) — contrast ratios, focus states, screen reader support
- **Touch & Interaction** (CRITICAL) — target sizes, gesture patterns, feedback
- **Performance** (HIGH) — image optimization, lazy loading, layout shift prevention
- **Style Selection** (HIGH) — matching style to product type, platform conventions
- **Layout & Responsive** (HIGH) — mobile-first, breakpoints, spacing systems
- **Typography & Color** (MEDIUM) — font pairing, semantic tokens, dark mode
- **Animation** (MEDIUM) — timing, physics-based curves, reduced-motion support
- **Forms & Feedback** (MEDIUM) — validation, error handling, empty states
- **Navigation** (HIGH) — tab bars, deep linking, state preservation
- **Charts & Data** (LOW) — chart type selection, accessible palettes, legends

Includes a pre-delivery checklist for visual quality, interaction, light/dark mode, layout, and accessibility.
