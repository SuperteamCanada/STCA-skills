# Solana AI Agent Scaffolder

A Claude skill that helps founders build production-ready AI agents on Solana using [SendAI's Agent Kit v2](https://github.com/sendaifun/solana-agent-kit) and the official [Solana Developer MCP](https://mcp.solana.com/).

## What the skill does

Guides you through scaffolding a complete Solana AI agent:

- **Plugin selection** — Token, NFT, DeFi, Blinks plugins based on your use case
- **Wallet setup** — Embedded wallet configuration via Turnkey/Privy for secure key management
- **MCP integration** — Configures Solana Developer MCP and Helius MCP for real-time on-chain queries
- **LLM wiring** — Integrates with Claude API, LangChain, or Vercel AI SDK
- **Devnet deployment** — Test agent actions before mainnet

## Why now

The Solana Foundation positions the network as core infrastructure for the agentic internet. 15M+ on-chain agent payments processed. Multiple frameworks (SendAI Agent Kit, ElizaOS, GOAT) compete in this space, with SendAI's modular plugin system as the most Solana-native option.
