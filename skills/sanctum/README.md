# Sanctum

A Claude skill for liquid staking on Solana: LST swaps, Infinity pool, staking, and liquidity provision.

## What the skill does

- Guides LST swap integration via Sanctum API (mSOL, jitoSOL, bSOL, INF, and more)
- Covers Infinity pool mechanics and reward-bearing INF token
- Provides staking flows, multi-hop routing, and batch operations
- Includes LST reference data, program IDs, and fee structures

## Why now

Liquid staking is the fastest-growing DeFi vertical on Solana. Sanctum unifies 20+ LSTs through a single swap interface and the INF reward-bearing token, making it the default staking aggregator.
