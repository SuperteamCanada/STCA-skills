# Metaplex

A Claude skill for NFT creation and management on Solana: MPL Core, Bubblegum (compressed NFTs), Candy Machine, Token Metadata, and DAS API.

## What the skill does

- Guides NFT creation using MPL Core (new standard) and legacy Token Metadata
- Covers compressed NFTs via Bubblegum with Merkle tree management
- Provides Candy Machine launch flows and Genesis token launches
- Includes Umi framework patterns, program IDs, fee schedules, and DAS API integration

## Why now

Metaplex is the dominant NFT standard on Solana. MPL Core is the recommended standard for new projects, and Bubblegum enables mass NFT distribution at 99% lower cost via state compression.
