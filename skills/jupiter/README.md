# Jupiter

A Claude skill for integrating Jupiter's DeFi APIs on Solana: swaps, perps, DCA, price feeds, token search, and more.

## What the skill does

- Guides integration of Jupiter Ultra Swap API for token swaps with gasless execution
- Covers Perps, Trigger orders, Recurring (DCA), Lend, and Portfolio APIs
- Provides production-ready error handling, rate limiting, and transaction patterns
- Includes endpoint reference, token list management, and troubleshooting

## Why now

Jupiter is the #1 DEX aggregator on Solana by volume. The Ultra API (gasless swaps) and expanded product suite (perps, lending, prediction markets) make it the default DeFi integration layer for wallets, agents, and dApps.
