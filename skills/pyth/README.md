# Pyth

A Claude skill for decentralized oracle price feeds on Solana: real-time prices, confidence intervals, EMA, and Anchor CPI integration.

## What the skill does

- Guides Hermes API integration for real-time and streaming price feeds
- Covers on-chain Anchor integration with PriceUpdateV2 accounts
- Provides confidence-aware pricing patterns and staleness validation
- Includes TypeScript client templates and Rust/Anchor oracle examples

## Why now

Pyth is the leading pull oracle on Solana with 500+ price feeds across crypto, equities, and commodities. Price-aware agents and DeFi integrations need reliable oracle data with confidence intervals.
