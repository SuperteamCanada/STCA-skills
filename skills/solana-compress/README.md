# ZK Compressed Token Launcher

A Claude skill for launching ZK compressed tokens on Solana using [Light Protocol](https://github.com/Lightprotocol/light-protocol), reducing on-chain storage costs by 99%.

## What the skill does

Guides you through building with ZK Compression:

- **Compressed token creation** — SPL and Token-2022 mints with compression
- **Mass distribution** — Airdrop to 100k+ wallets for pennies
- **Token-2022 extensions** — Transfer hooks, transfer fees, permanent delegation under compression
- **Cost calculator** — Compare compressed vs uncompressed costs for your scale

## Why now

Light Token SDK is live on mainnet. Token-2022 extensions now work with compression. Any operation that creates many token accounts — airdrops, loyalty programs, gaming assets, agent micropayments — becomes 99% cheaper.
