# Solana Blinks & Actions Builder

A Claude skill that helps founders build [Solana Actions](https://solana.com/developers/guides/advanced/actions) and wrap them as [Blinks](https://www.dialect.to/) — shareable transaction URLs that render natively inside X (Twitter), Phantom, and other clients.

## What the skill does

Guides you through building a complete Blinks integration:

- **Action API scaffold** — GET endpoint for metadata, POST endpoint that returns unsigned transactions
- **Common patterns** — Token swaps, NFT mints, tipping, voting, donations
- **Dialect registry** — Registration for verified Blink status on X
- **Testing** — End-to-end validation with the Blinks Inspector

## Why now

X Money launched in April 2026 with native Blinks support. Users can execute swaps, payments, and mints directly inside X without leaving the app. This is the biggest distribution channel unlock in crypto.
