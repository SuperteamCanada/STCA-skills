# Solana Developer Platform (SDP) Starter

A Claude skill for building enterprise payment, tokenization, and trading products on the [Solana Developer Platform](https://solana.com/solutions/sdp).

## What the skill does

Guides you through building on SDP's three modules:

- **Issuance** — Tokenized deposits, stablecoins, RWAs with Token-2022 extensions
- **Payments** — Fiat/stablecoin orchestration with on/off-ramp partners
- **Trading** — Swaps, vaults, and on-chain FX via aggregated liquidity

Covers compliance integration (KYC/AML), sandbox setup, and infrastructure partner configuration.

## Why now

SDP launched March 24, 2026 with Mastercard, Worldpay, and Western Union as early adopters. It aggregates 20+ infrastructure providers into a single API and is explicitly designed to work with Claude Code.
