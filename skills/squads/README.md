# Squads

A Claude skill for smart accounts and multisig on Solana: Squads V4, session keys, spending limits, and Grid stablecoin payments.

## What the skill does

- Guides multisig creation with configurable permissions, thresholds, and time locks
- Covers smart account features: session keys, spending limits, account abstraction
- Provides Grid API integration for stablecoin payments and virtual accounts
- Includes vault transaction patterns, proposal workflows, and treasury management

## Why now

Squads is the emerging standard for programmable accounts on Solana. Smart accounts enable session keys and spending limits — critical infrastructure for wallets, DAOs, and AI agents that need scoped transaction authority.
