# Helius

A Claude skill for Solana RPC and API infrastructure: DAS API, webhooks, priority fees, Enhanced Transactions, and LaserStream.

## What the skill does

- Guides DAS API integration for NFT/token indexing (getAsset, searchAssets, getAssetsByOwner)
- Covers priority fee estimation and optimized transaction sending
- Provides webhook setup with deduplication patterns
- Includes WebSocket streaming, Enhanced Transactions, and ZK Compression support

## Why now

Helius is Solana's leading RPC and API platform, powering major wallets and dApps. The DAS API is the standard for asset indexing, and priority fee optimization is critical for reliable transaction landing.
