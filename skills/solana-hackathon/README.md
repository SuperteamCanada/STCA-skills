# Solana Hackathon Project Bootstrapper

A Claude skill that fast-tracks Solana hackathon submissions — from idea to working dApp with the right 2026 primitives.

## What the skill does

- **Guided setup** — Pick your category, get recommended primitives (ZK Compression, Blinks, Agent Kit, SDP, Token-2022)
- **Full scaffold** — Anchor program + Next.js frontend + wallet adapter + devnet deployment
- **Submission package** — README template, demo video checklist, pitch video (pairs with `winning-pitch-deck` skill)
- **Time management** — Scope recommendations based on days remaining

## Why now

Colosseum Frontier Hackathon runs April 6 – May 11, 2026. $250K pre-seed for accelerator winners. Multiple concurrent Solana hackathons run year-round.

## Pairs with

| Skill | When to use |
|-------|-------------|
| `winning-pitch-deck` | Build the 3-min pitch video |
| `solana-agent` | AI agent projects |
| `solana-blinks` | Shareable transactions on X |
| `solana-compress` | Mass token distribution |
| `solana-sdp` | Payments / tokenization |
| `ui-ux-pro-max` | Make the frontend look professional |
