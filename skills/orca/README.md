# Orca

A Claude skill for concentrated liquidity on Solana: Whirlpools AMM, swaps, position management, and fee harvesting.

## What the skill does

- Guides Whirlpools SDK integration for swaps with slippage control
- Covers concentrated liquidity position management (open, close, rebalance)
- Provides pool creation, fee harvesting, and reward collection patterns
- Includes setup templates, program addresses, and migration guidance (legacy → new SDK)

## Why now

Orca's Whirlpools is the leading concentrated liquidity AMM on Solana. The new SDK (Web3.js v2 based) offers tree-shakeable, type-safe integration for wallets, agents, and DeFi frontends.
